# 功夫大师 - 横版街机通关游戏

## 运行方式
直接双击 `kungfu-fighter.html` 或在浏览器中打开即可运行。无需服务器。

## 操作说明
- **A / D**：左右移动
- **空格**：跳跃
- **J**：拳击（咏春寸劲）- 冷却短，伤害 25
- **K**：腿击（洪拳虎鹤）- 冷却中，伤害 35
- **L**：大招（爆发技）- 需蓄满能量，伤害 80，范围大

## 数值设计

### 主角
- 生命：100
- 拳击：25 伤害，冷却 25 帧
- 腿击：35 伤害，冷却 40 帧
- 大招：80 伤害，冷却 180 帧，蓄满能量约 8 秒

### 敌人防御
- 普通小兵：防御 0.75–0.9（减伤 10–25%）
- Boss：防御 0.45–0.6（减伤 40–55%）

### 难度曲线
- 每关血量递增：800 → 1000 → 1200 → 1500
- 敌人生成间隔随关卡缩短
- 过关后恢复 30 生命

## 自定义 BGM
将 `kungfu-bgm.mp3` 放在与 HTML 同目录下，游戏会自动使用该文件作为背景音乐。若无此文件，游戏会使用内置合成器音效。

## 设计原则
- **可玩性**：拳/腿/大招组合，跳跃可躲避攻击
- **挑战性**：Boss 血厚防高，需合理使用大招
- **不过度**：无敌帧 45 帧，过关回血，蓄力可随时间恢复
